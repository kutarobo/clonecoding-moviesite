    - 로컬 개발용 환경파일 추가 (server/dev.js)
3. 몽고디비 세팅
    - key 가 필요, 몽고db에 로그인해서 받을 수 있다. [mongodb.com](mongodb.com)
    - 로그인 -> 클러스터 생성
        - aws 및 region 선택
        - M0 (free tier) 선택
        - cluster name 지정 후 create cluster
        - connect your application
            - add user
            - add fire wall ip
            - key copy -> dev.js 복붙
